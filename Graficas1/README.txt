In C: disk must be a "Externals" folder. 
Inside are the "Assimp" (5.0.1), "FreeImage" (3.18), "GLM" (0.9.9.9), "ImGui" (1.8) and "OpenGL"
folder which contain their respective include files.
The "OpenGL" folder must contain a "GLFW" (3.3.3) folder and "glad" (3.3) folder.

QE - Zoom in/out camera
